### Example

Open a dos prompt and type:


```dos

            cmd.exe /c "C:\nodejs\npm.cmd" install -g jsdoc
        
```


or:


```dos

            powershell -c cmd.exe /c "C:\nodejs\npm.cmd" install -g jsdoc
        
```
